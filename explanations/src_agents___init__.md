# `src/agents/__init__.py`

## What this file does
This file is intentionally empty. It marks `src/agents` as a Python package.

## Why it exists
An `__init__.py` file tells Python that the folder can be imported as a module package, for example:

```python
from agents.pg_agent import PGAgent
```

## In plain words
This file does not contain logic. It just helps Python understand that `agents` is a package.
